Smart Chatter UI

Smart Chatter UI improves productivity by letting you hide and show chatter in any Odoo form view. A floating "Show Chatter" button ensures you can bring it back anytime.

Key Features

  • Hide chatter instantly
  • Floating "Show Chatter" button
  • Works across Sales Orders, Invoices, CRM, Employees & more
  • Clean, lightweight, and non-intrusive

Usage Guide

  1. Open a form (e.g., Sales Order, Invoice, CRM Lead, Employee)
  2. Click "Hide" in the chatter
  3. Click floating "Show Chatter" to restore it
